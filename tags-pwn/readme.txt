1. Rode o arquivo sorteio trocando a variavel DRE pelo seu proprio DRE para descobrir quais desafios voce tem q fazer
2. Exploite os desafios. Tente conclui-los com o ASLR habilitado em seu sistema. Nao deve ser necessario usar força bruta
3. Se estiver muito dificil, peça ajuda.
4. Se estiver muito dificil, tente desabilitar o ASLR. Voce pode também inverter a ordem e tentar com o ASLR desabilitado primeiro para facilitar.
5. Se não for possivel fazer, ao menos escreva no writeup o por que estava dificil e quais tecnicas você tentou utilizar.
6. Deixa o writeup no seu git mesmo.

Obs:
 Para essa tag eu selecionei alguns desafios simples presentes em CTFs internacionais de nível variado.
 Para os desafios ez eu ocultei o nome pra evitar copia e cola de writeup ja existentes, depois da entrega eu revelo o nome dos challs pra commitar bonitinho no git.
